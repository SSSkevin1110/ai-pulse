# AI Pulse 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位
AI Pulse 是一个专注于 AI 领域的资讯聚合平台，致力于为用户提供最新、最全的人工智能行业动态、技术进展和产品发布信息。

### 1.2 目标用户
- AI 从业者（开发者、研究员、产品经理）
- 科技爱好者和早期采用者
- 投资人和行业分析师
- 对 AI 感兴趣的大众用户

### 1.3 核心价值主张
- **及时性**: 7×24小时追踪全球 AI 资讯
- **全面性**: 覆盖 LLM、机器人、自动驾驶等多领域
- **智能性**: 个性化推荐，精准匹配用户兴趣
- **美观性**: 简洁大气的视觉体验

## 2. 设计规范

### 2.1 视觉风格
参考 MiniMax 中国站风格：
- **设计语言**: 简洁、大气、科技感
- **核心特征**: 玻璃态设计 (Glassmorphism)、动态背景
- **布局风格**: 卡片式布局、大留白、呼吸感

### 2.2 色彩系统

#### 主色调
| 颜色 | 色值 | 用途 |
|------|------|------|
| 品牌红 | `#E53935` | 主按钮、高亮、标签、强调元素 |
| 纯白 | `#FFFFFF` | 背景、卡片底色、文字 |
| 深红 | `#C62828` | 悬停状态、Pressed 状态 |
| 浅红 | `#FFEBEE` | 浅色背景、标签背景 |

#### 辅助色
| 颜色 | 色值 | 用途 |
|------|------|------|
| 深灰 | `#1A1A1A` | Dark 模式背景 |
| 中灰 | `#6B7280` | 次要文字、图标 |
| 浅灰 | `#F3F4F6` | Light 模式背景、分割线 |
| 边框灰 | `#E5E7EB` | 边框、分割线 |

#### 玻璃态效果
```css
/* Light 模式玻璃态 */
background: rgba(255, 255, 255, 0.85);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.3);
box-shadow: 0 8px 32px rgba(229, 57, 53, 0.1);

/* Dark 模式玻璃态 */
background: rgba(26, 26, 26, 0.85);

border: 1px solid rgba(255, 255, 255, 0.1);
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3);
```

- **标题字体**: system-ui, -apple-system, sans-serif
- **正文字体**: system-ui, -apple-system, sans-serif

## 4. 新闻卡片设计

### 4.1 卡片字段定义

#### 核心字段
| 字段名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| `id` | String | 是 | 唯一标识符 |
| `title` | String | 是 | 标题 (最多 80 字) |
| `summary` | String | 是 | 摘要 (最多 200 字) |
| `content` | String | 否 | 完整内容 (Markdown) |
| `source` | Object | 是 | 来源信息 |
| `source.name` | String | 是 | 来源名称 |
| `source.url` | String | 是 | 原文链接 |
| `source.favicon` | String | 否 | 来源图标 |
| `coverImage` | String | 否 | 封面图 URL |
| `tags` | Array | 是 | 标签列表 |
| `publishedAt` | DateTime | 是 | 发布时间 |
| `readTime` | Number | 否 | 阅读时长 (分钟) |
| `viewCount` | Number | 否 | 浏览数 |
| `likeCount` | Number | 否 | 点赞数 |
| `isFeatured` | Boolean | 否 | 是否精选 |

#### 扩展字段
| 字段名 | 类型 | 说明 |
|--------|------|------|
| `author` | Object | 作者信息 (AI 生成/人工编辑) |
| `aiSummary` | String | AI 生成的摘要 |
| `relatedArticles` | Array | 相关文章 ID |
| `language` | String | 语言 (zh/en) |
| `isOriginal` | Boolean | 是否原创 |

### 4.2 卡片展示逻辑

#### 卡片类型

**1. featured-card (精选大卡片)**
```
┌─────────────────────────────────────┐
│  [封面图 - 16:9]                     │
│                                     │
├─────────────────────────────────────┤
│  🔴 标签1  标签2                     │
│  标题文字 (最多2行)                  │
│  摘要文字 (最多3行)...               │
│                                     │
│  🏢 来源名称  ·  2小时前  ·  5分钟   │
└─────────────────────────────────────┘
```
- 尺寸: 全宽
- 使用场景: 首页 Hero 区域、编辑推荐
- 封面图比例: 16:9
- 标题: 最多 2 行
- 摘要: 最多 3 行

**2. standard-card (标准卡片)**
```
┌─────────────────────────────────────┐
│  [封面图 - 4:3]    │  🔴 标签1       │
│                   │  标题文字       │
│                   │  摘要文字...    │
│                   │                 │
│                   │  🏢 来源 · 时间 │
└─────────────────────────────────────┘
```
- 尺寸: 双列布局
- 封面图比例: 4:3 (左侧 40%)
- 使用场景: 资讯流默认展示
- 标题: 最多 2 行
- 摘要: 最多 2 行

**3. compact-card (紧凑卡片)**
```
┌─────────────────────────────────────┐
│  🔴 标签1  标签2                     │
│  标题文字 (最多2行)                  │
│  🏢 来源名称  ·  2小时前             │
└─────────────────────────────────────┘
```
- 尺寸: 全宽，无封面图
- 使用场景: 列表模式、搜索结果
- 标题: 最多 2 行
- 无摘要

**4. mini-card (迷你卡片)**
```
┌─────────────────────────────────────┐
│  🔴 标签    标题文字...    2小时前   │
└─────────────────────────────────────┘
```
- 尺寸: 全宽，单行
- 使用场景: 侧边栏推荐、相关文章
- 标题: 最多 1 行

### 6.3 动态背景效果

#### Light 模式背景
```
渐变基底: linear-gradient(135deg, #FFFFFF 0%, #FFF5F5 50%, #FFEBEE 100%)
动态元素: 
  - 漂浮的红色光晕 (brand-red, 5% opacity)
  - 缓慢移动的玻璃态圆形
  - 微妙的噪点纹理
```

#### Dark 模式背景
```
渐变基底: linear-gradient(135deg, #0A0A0A 0%, #1A0A0A 50%, #2A1A1A 100%)
动态元素:
  - 深色背景上的红色光晕 (brand-red, 10% opacity)
  - 星点粒子效果
  - 玻璃态卡片的发光边框
```

### 6.4 主题切换交互

#### 切换入口
```
设置页面 / 导航栏
┌─────────────────────────────────────┐
│  外观                               │
│  ─────────────────────────────────  │
│  ☀️  Light                     ○   │
│  🌙  Dark                      ○   │
│  🔄  跟随系统                   ●   │
└─────────────────────────────────────┘
```

#### 切换动画
- **过渡时长**: 300ms
- **缓动函数**: cubic-bezier(0.4, 0, 0.2, 1)
- **动画效果**: 
  - 背景色渐变过渡
  - 文字颜色渐变过渡
  - 卡片阴影调整
  - 玻璃态效果保持

#### 系统跟随逻辑
```javascript
// 监听系统主题变化
const mediaQuery = window.matchMedia('(prefers-color-scheme: dark)');

if (themeMode === 'auto') {
  if (mediaQuery.matches) {
    applyDarkTheme();
  } else {
    applyLightTheme();
  }
}

// 系统变化时自动更新
mediaQuery.addEventListener('change', (e) => {
  if (themeMode === 'auto') {
    e.matches ? applyDarkTheme() : applyLightTheme();
  }
});
```

### 6.5 主题持久化

- **存储方式**: localStorage
- **存储键**: `ai-pulse-theme`
- **存储值**: `light` | `dark` | `auto`
- **默认值**: `auto`

## 8. 交互细节

### 8.1 动画规范

| 动画类型 | 时长 | 缓动 | 用途 |
|----------|------|------|------|
| 页面切换 | 300ms | ease-out | 路由切换 |
| 卡片悬停 | 200ms | ease | 卡片交互 |
| 按钮点击 | 100ms | ease-in-out | 按钮反馈 |
| 弹窗出现 | 250ms | cubic-bezier(0.4, 0, 0.2, 1) | 弹窗动画 |
| 骨架屏 | 1.5s | ease-in-out | 加载状态 |
| 主题切换 | 300ms | ease | 主题变化 |

### 8.2 手势操作

| 手势 | 场景 | 效果 |
|------|------|------|
| 下拉刷新 | 资讯流 | 刷新列表 |
| 上拉加载 | 资讯流 | 加载更多 |
| 左滑 | 资讯卡片 | 快速分享 |
| 右滑 | 资讯卡片 | 快速收藏 |
| 长按 | 资讯卡片 | 弹出操作菜单 |
| 双击 | 图片 | 放大查看 |

## 9. 技术实现建议

### 9.1 前端技术栈
- **框架**: Next.js 14 + React 18
- **样式**: Tailwind CSS + CSS Variables
- **动画**: Framer Motion
- **图标**: Lucide React
- **字体**: Inter (英文) + 系统字体 (中文)

### 9.2 响应式断点
| 断点 | 宽度 | 布局 |
|------|------|------|
| Mobile | < 640px | 单列 |
| Tablet | 640px - 1024px | 双列 |
| Desktop | > 1024px | 三列 |

### 9.3 性能优化
- 图片懒加载 (next/image)
- 虚拟滚动 (长列表)
- 骨架屏预加载
- 字体预加载
- CSS 变量主题切换 (无闪烁)
