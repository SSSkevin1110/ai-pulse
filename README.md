## 📦 Getting Started | 开始使用

### Prerequisites | 前置条件

- Node.js 18+ 
- npm or yarn

### Installation | 安装

```bash
# Clone the repository | 克隆仓库
git clone https://github.com/SSSkevin1110/ai-pulse.git

# Navigate to project | 进入项目目录
cd ai-pulse

# Install dependencies | 安装依赖
npm install

# Start development server | 启动开发服务器
npm run dev
```

The site will be available at `http://localhost:4321`

网站将在 `http://localhost:4321` 可用

## 🚀 Deployment | 部署

### GitHub Pages (Recommended) | 推荐

1. Push code to GitHub
2. Go to Settings → Pages
3. Select "GitHub Actions" as source
4. The site will auto-deploy on every push

每次推送到 main 分支时自动部署

### Manual Build | 手动构建

```bash
# Build for production | 生产构建
npm run build

# Preview production build | 预览生产构建
npm run preview
```
